What is this app about?

Finance Book

This is a simple app to track the money people owe to you

This app contains a Register User page, Login Page, main page where you are asked to select the options present which are 'Create User', 'Show Users' and a logout button

Login

Only registered users can login

Register

Register a new user

Create User

Contains a text field and the money field to add the user and money to a table which is stored in a database

Show Users

Contains an HTML table which contains the users, money they owe, Update and Delete buttons which completes the CRUD features
